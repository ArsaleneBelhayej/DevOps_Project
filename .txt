you should create a configmap 
----- example ----
apiVersion: v1
kind: ConfigMap
metadata:
  name: config-server-config
  namespace: webapps
data:
  SPRING_CLOUD_CONFIG_SERVER_GIT_URI: https://github.com/samiouerfelli/DEVOPS_PROJECT_G2.git
  SPRING_CLOUD_CONFIG_SERVER_GIT_DEFAULT_LABEL: config-server  # Specify the Git branch here
 
------ 
( login details of github)


echo -n 'your-username' | base64
echo -n 'your-password' | base64

----
then creta a secret file 

-----
apiVersion: v1
kind: Secret
metadata:
  name: config-server-secrets
  namespace: webapps
type: Opaque
data:
  SPRING_CLOUD_CONFIG_SERVER_GIT_USERNAME: <base64-encoded-username>
  SPRING_CLOUD_CONFIG_SERVER_GIT_PASSWORD: <base64-encoded-password>
